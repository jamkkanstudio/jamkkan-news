# Sprint 01 — MVP와 성장 기반 구축

## 상태

완료 · 2026-07-14~2026-07-15

이 문서는 당시 별도 Sprint 채팅이 없었기 때문에 Git 이력과 현재 문서를 기준으로 복원한 회고 기록입니다.

## 목표

30초 안에 중요한 뉴스를 확인하고, 읽은 시간을 성장으로 기록할 수 있는 Jamkkan MVP를 배포합니다.

## 완료한 기능

- 뉴스 직접 추가·수정·삭제
- 중요도 기반 오늘의 TOP5
- 관심 분야 기반 나의 TOP5
- RSS 뉴스 수집과 최대 5개 선택 등록
- 규칙 기반 짧은 브리핑과 중요 이유 생성
- 기사별 30초 투자 완료 기록
- 오늘·누적 투자 시간과 연속 이용 기록
- 성장 배너와 나의 성장 페이지
- 브리핑 완료 화면
- Daily Goal 설정과 진행률
- 기사 읽기 이벤트와 나의 분석
- 주간·카테고리별 읽기 통계
- Streamlit MVP 배포

## 주요 제품 결정

- 많이 보여주기보다 중요한 뉴스에 집중합니다.
- 체류시간 대신 사용자가 자신에게 투자한 시간을 핵심 지표로 봅니다.
- 무한 스크롤과 중독형 UX를 만들지 않습니다.
- 초기 MVP 데이터는 JSON 파일을 기준으로 저장합니다.
