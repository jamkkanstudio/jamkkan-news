## 프로젝트 소개

Jamkkan News는 매일 아침 가장 중요한 뉴스를 선별하여
짧고 명확하게 전달하는 뉴스 큐레이션 서비스입니다.

뉴스를 많이 보여주는 것이 아니라,

**가장 중요한 뉴스만 전달하는 것**을 목표로 합니다.

---

## 프로젝트 목표

- 하루 핵심 뉴스 5개 제공
- AI 기반 뉴스 요약
- 뉴스 중요도 평가
- 뉴스 자동 수집
- 아침 뉴스 브리핑 서비스
